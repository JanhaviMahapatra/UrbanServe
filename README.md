# AI Service Marketplace

A full-stack marketplace where users can book home services and providers manage bookings.

## Features

- User authentication
- Provider onboarding
- Service management
- Booking system
- Razorpay payments
- AI service assistant (DeepSeek via OpenRouter)
- Floating chatbot
- Provider dashboard

## Tech Stack

Frontend
- React
- Axios
- React Router

Backend
- Node.js
- Express
- Prisma
- PostgreSQL

AI
- OpenRouter
- DeepSeek

Payments
- Razorpay
